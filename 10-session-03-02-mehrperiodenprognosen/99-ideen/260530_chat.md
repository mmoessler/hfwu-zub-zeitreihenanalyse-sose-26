  Die iterierte Mehrperiodenprognose mit einem AR-Modell wird schrittweise berechnet.
  
  Zuerst wird die Einperiodenprognose berechnet, anschließend daraus die Zweiperiodenprognose usw.
  
  Die iterierten Zwei- und Dreiperiodenprognosen mit einem AR($p$)-Modell lauten:
  
  \vspace{-0.25cm}
  
  \begin{align*}
  \hat{Y}_{T+2|T} &= \hat{\beta}_0 + \hat{\beta}_1 \hat{Y}_{T+1|T} + \hat{\beta}_2 Y_T + \hat{\beta}_3 Y_{T-1} + \cdots + \hat{\beta}_p Y_{T-p+2} \\
  \hat{Y}_{T+3|T} &= \hat{\beta}_0 + \hat{\beta}_1 \hat{Y}_{T+2|T} + \hat{\beta}_2 \hat{Y}_{T+1|T} + \hat{\beta}_3 Y_T + \cdots + \hat{\beta}_p Y_{T-p+3} \\
  \cdots
  \end{align*}
  
  \vspace{-0.25cm}
  
  Dabei sind die $\hat{\beta}$ die OLS-Schätzwerte der AR($p$)-Koeffizienten.

\section*{Direkte Mehrperioden-Prognosen}



%..............................................................................
\begin{frame}
  \frametitle{Direkte Mehrperioden-Prognosen \hfill (1/2)}

  Was sind direkte Mehrperioden-Prognosen?
  
  \begin{itemize}
    \item Direkte Prognosen werden ohne Iteration berechnet.
    \item Eine Regression schätzt direkt $Y_{T+h}$ mit verzögerten Regressoren.
    \item Die Schätzkoeffizienten können sofort für die Prognose verwendet werden.
  \end{itemize}  

  Beispiel: Direkte Zwei-Perioden-Prognose
  
  \vspace{-0.50cm}
  
  \begin{align*}
    \text{GDPGR}_{t|t-2} &= 0{,}56 + 0{,}31 \cdot \text{GDPGR}_{t-2} + 0{,}04 \cdot \text{GDPGR}_{t-3} \\
    &+ 0{,}56 \cdot \text{TSpread}_{t-2} + 0{,}04 \cdot \text{TSpread}_{t-3}
  \end{align*}

  \vspace{-0.25cm}

  \begin{itemize}
    \item Prognose für 2018:Q1 basierend auf 2017:Q3 ergibt ca. 2{,}4\%.
  \end{itemize}

  Wichtig
  
  \begin{itemize}
    \item Mehrperioden-Fehler sind seriell korreliert.
    \item $\Rightarrow$ Verwende Newey-West HAC-Standardfehler.
  \end{itemize}  

\end{frame}

  \frametitle{Direkte Mehrperioden-Prognosenn \hfill (2/2)}

  \metroset{block=fill}

  \begin{alertblock}{Schl\"usselkonzept: Direkte Mehrperioden-Prognosen}

  Die direkte Mehrperioden-Prognose $h$ Perioden in die Zukunft basierend auf $p$ Verzögerungen von $Y_t$ und einem zusätzlichen Prädiktor $X_t$ erfolgt durch die Schätzung der folgenden Regression:

  \[
  Y_t = \delta_0 + \delta_1 Y_{t-h} + \cdots + \delta_p Y_{t-p-h+1} + \delta_{p+1} X_{t-h}
  + \cdots + \delta_{2p} X_{t-p-h+1} + u_t
  \]
  
  und anschließend durch direkte Verwendung der geschätzten Koeffizienten zur Prognose von $Y_{T+h}$ mit Daten bis zur Periode $T$.
  
	\end{alertblock}

\begin{frame}
  \frametitle{Welche Methode sollten Sie verwenden? \hfill (1/2)}

  Zwei Methoden zur Mehrperiodenprognose
  
  1. Iterative Methode:
  \begin{itemize}
    \item Prognose durch wiederholte Anwendung des Ein-Perioden-Modells (z.B. AR oder VAR).
    \item Verwendung desselben Modells für jede Prognosehorizont.
  \end{itemize}
  
  1. Direkte Methode:
  \begin{itemize}
    \item Direkte Schätzung eines Modells für die Prognose $h$ Perioden im Voraus.
    \item Unterschiedliches Modell für jeden Prognosehorizont.
  \end{itemize}  

\end{frame}

\begin{frame}
  \frametitle{Welche Methode sollten Sie verwenden? \hfill (2/2)}

  Warum meistens die iterative Methode?
  
  \begin{itemize}
    \item Theoretisch effizienter: Wenn das Ein-Perioden-Modell korrekt spezifiziert ist, liefert die iterative Methode effizientere Schätzungen.
    \item Praktisch stabiler: Gleiche Modelle für verschiedene Horizonte führen zu gleichmäßigeren Zeitverläufen der Prognosen.
    \item Weniger Zufallsschwankungen: Direkte Methode führt durch Schätzfehler eher zu unruhigen Prognoseverläufen.
  \end{itemize}
  
  Wann ist die direkte Methode besser?
  
  \begin{itemize}
    \item Wenn das Ein-Perioden-Modell nicht korrekt spezifiziert ist.
    % \item Beispiel: Falsche Spezifikation eines oder mehrerer Gleichungen in einem VAR-Modell (z.B. vernachlässigte Nichtlinearitäten).
    \item Folge: Iterative Methode kann verzerrte und ineffiziente Prognosen liefern.
    \item Direkte Methode hat in solchen Fällen potenziell geringeren mittleren quadratischen Prognosefehler (MSFE).
  \end{itemize}  

\end{frame}
